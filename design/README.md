# CedarEx 工具台交互设计图

## 设计目标

- 按非代码用户使用习惯优化为 CedarEx 工具台。
- 保留必要界面结构：工具列表、中心工作区、智能助手、底部执行过程、状态栏。
- 顶部标题栏只保留窗口控制点，删除命令搜索和环境信息，减少视觉噪音。
- 底部区域不再强调终端概念，只保留单一执行过程、状态提示和记录操作。
- 界面文案尽量使用中文，减少脚本、参数、终端、Output、AI Review 等偏技术表达对非代码用户的干扰。
- `脚本说明` 只放在 `使用说明` 标签页中，右侧智能助手不重复展示说明文档。
- HTML 设计稿支持基础交互：切换标签、选择工具、填写信息、模拟运行、清空记录和触发助手快捷动作。
- 运行完成后在中心区域展示主结果卡片，避免用户必须阅读底部日志。

## 设计图文件

- `vscode-script-workbench.html`：1440x900 交互 HTML 设计图。

## 布局说明

1. 工具列表：显示搜索框、工具数量、刷新入口和常用工具列表。
2. 中心工作区：包含 `填写信息`、`使用说明` 两个可切换标签。
3. 智能助手：只展示现在可以做什么、一键帮我和运行前确认，不展示脚本说明。
4. 执行过程：底部展示执行过程、状态提示和记录操作按钮。
5. 状态栏：展示 CedarEx 工具台、智能助手状态和编码信息。
