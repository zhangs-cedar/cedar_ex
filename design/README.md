# CedarEx 当前效果设计图

## 设计目标

- 按当前实现截图重生成 VSCode 风格脚本工作台空状态。
- 保留必要界面结构：Activity Bar、脚本资源管理器、编辑区标签、AI Copilot Panel、Terminal Panel、Status Bar。
- 顶部标题栏只保留窗口控制点，删除命令搜索和环境信息，减少视觉噪音。
- 底部终端不再显示红黄绿装饰按钮、`CedarEx - 空闲中` 和 `真实终端` 等技术状态，只保留标签、操作按钮和日志内容。
- 强调当前状态：未加载脚本、中心引导选择脚本、终端处于 idle、AI 快捷动作可见。
- 界面文案尽量使用中文，减少 README、Terminal、Output、AI Review 等偏技术英文对非代码用户的干扰。
- `脚本说明` 只放在 `使用说明` 标签页中，右侧智能助手不重复展示说明文档。
- HTML 设计稿支持基础交互：切换标签、选择脚本、填写参数、模拟运行、清空日志和触发助手快捷动作。

## 设计图文件

- `vscode-script-workbench.html`：1440x900 静态高保真 HTML 设计图。

## 布局说明

1. 左侧入口栏：脚本、运行、搜索、辅助入口和智能助手固定入口。
2. Explorer：显示脚本资源管理器、搜索框、脚本数量和刷新入口。
3. 中心工作区：包含 `填写参数`、`使用说明`、`运行记录` 三个可切换标签。
4. 智能助手：只展示下一步建议、快捷动作和运行前检查，不展示脚本说明。
5. 底部终端：底部真实终端面板，包含终端、问题、输出、智能分析标签和日志操作按钮。
6. 状态栏：展示 CedarEx 工作区、智能助手状态和编码信息。
